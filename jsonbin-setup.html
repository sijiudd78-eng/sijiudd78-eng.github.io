🌍 JSON Bin Setup - Global User Management

This guide will help you set up JSON Bin so you can see users from ALL countries.

📋 What This Does:
• Users from USA, UK, India, etc. all save to SAME database
• You (Admin) can see ALL users globally
• Real-time updates across all devices
• 100% free with generous limits

Step 1: Create JSON Bin Account

1. Go to https://jsonbin.io

2. Sign up for free account

3. Verify your email

Step 2: Create Your First Bin

1. Click "Create New Bin" button

2. Give it a name like: shopzene-users

3. Set visibility to "Private"

4. Click "Create Bin"

Step 3: Get Your Credentials

You will see TWO important codes:

Bin ID: Your bin ID (looks like: 65f8a1c9-9b4c-4e8d-8c7a-1a2b3e4f6a7)

Master Key: Your secret key (long string of characters)

Step 4: Update Your Code

Replace these TWO lines in your files:

In login.html and admin-new.html:
const JSON_BIN_ID = 'your-bin-id-here'; // Replace with your actual bin ID
const JSON_BIN_KEY = 'your-master-key-here'; // Replace with your actual master key

Step 5: Test Global Users

1. Deploy updated files to Netlify

2. Have friends from different countries sign up

3. Log in as admin - you should see ALL users!

⚠️ Important: Keep your Master Key secret! Anyone with your key can access all user data.
